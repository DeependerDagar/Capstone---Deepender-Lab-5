# Capstone-Deepender-Lab-5
 # Travel Agency Website - Capstone Project

 ## Project Title & Theme
Travel Agency Website - A responsive website for showcasing travel destinations, services, and booking options.

 ## Sections Implemented
- Home page with featured destinations and promotions  
- Services section outlining travel packages and custom tours  
- Booking form for user inquiries and reservations  
- Gallery with images of popular tourist spots  

 ## Breakpoints and Layout Techniques
- Breakpoints at 480px, 768px, and 1024px for mobile, tablet, and desktop views  
- Flexbox and CSS Grid used for main layout and responsive design  
- Media queries to adapt navigation, images, and content layout across devices
